Structural Elements

Panel

A panel is a block element that has the ability to load data asycronously when a src attribute is set (same-origin policy restricted) 


<x-panel src="demo/html/panel-content.html"></x-panel>
Tab Box

A container used to display a set of tabbed pages of elements. A row of tabs is displayed at the top of tabbox which may be used to switch between each page. The tabbox may contain a tabs element, a cointainer for tab elements, and a tabpanels element, which serves as a contianer for tabpanel elements.

Tab 1 Tab 2 Tab 3

Tabboxes are great for showing users only the conent they need to see, while providing users an easy way to access additional content on-demand

Examples of use-cases for tabboxes include: a set of step-wise directions, grouping like content elements together, or as a building block for a tabbed viewing interface in a mobile web app.

In short, tabboxes are pretty freaking cool, and when you pair them with CSS animations/transitions, you can do things with a few styles that used to take a ton of JavaScript and custom code. 


<x-tabbox>
	<x-tabs>
		<x-tab selected="true">Tab 1</x-tab>
		<x-tab>Tab 2</x-tab>
		<x-tab>Tab 3</x-tab>
	</x-tabs>
	
	<x-tabpanels>
		<div selected="true">
			<p>ONE</p>
		</div>
		
		<div>
			<p>TWO</p>
		</div>

		<div>
			<p>THREE</p>
		</div>
	</x-tabpanels>
</x-tabbox>

Media Elements

Map

The map element automatically loads a Leaflet/OpenStreet map centered on either the user's location, a set of lat/long coordinates provided via attribute, or an internal default (Montain View, CA - had to pick somewhere!)

Input Elements

Autosuggest

An autosuggest input provides automated hints and completion of textual value based on keyboard input.

Navigation Elements

Pager

A pager element that allows you set properties and it will do the rest

Action Bar

An iconified navigation that fire user-defined actions.

Compositions

App View

The appview element is the container for arranging content a common app layout (similar to Android and iOS). The element responsively transitions the app to an optimal layout state based on orientation and screen size and manages the location of global menus and actions.
