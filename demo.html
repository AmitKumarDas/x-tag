Tabbox

  • Tabbox

    A container used to display a set of tabbed pages of elements. A row of tabs is displayed at the top of tabbox which may be used to switch between each page. The tabbox may contain a tabs element, a cointainer for tab elements, and a tabpanels element, which serves as a contianer for tabpanel elements.

    Tab 1 Tab 2 Tab 3

    Tabboxes are great for showing users only the conent they need to see, while providing users an easy way to access additional content on-demand

    Examples of use-cases for tabboxes include: a set of step-wise directions, grouping like content elements together, or as a building block for a tabbed viewing interface in a mobile web app.

    In short, tabboxes are pretty freaking cool, and when you pair them with CSS animations/transitions, you can do things with a few styles that used to take a ton of JavaScript and custom code.
