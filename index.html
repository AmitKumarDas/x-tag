Intro

What it Does

When you need an advanced HTML element for your app - like a tabbox, tree, special input, etc. - you're probably familiar with this process:

  1. find a plugin
  2. add the plugin script include to your page
  3. paste an element, or block of elements, into your markup
  4. add a script block or external script include for your own code
  5. write code to select elements on page load
  6. instantiate a script to augment the existing vanilla markup
  7. wash, rinse, repeat for every plugin you need

But what if all you had to do was:

  1. Drop an HTML tag into the page
  2. There is no step 2, it all Just Works™

Welcome to X-Tag folks.

How it Works

In order to allow for the recognition and parsing of custom HTML tags, X-Tag utilizes a special CSS event mechanism to listen for the creation/parsing of new custom elements (details here). This even is fired and processesed by x-tag regardless of whether the element is present in the source of the document or if you use document.createElement to generate it dynamically. When the node is initially created x-tag adds events and calls the onCreate function on the node, if you've registered them for that tag. You can also specify an action should be run each time a specific type of element is inserted into the DOM with onInsert. Here's an example with some comments for a bit more clarity: 


mozComponents.register('accordion', {
	onCreate: function(){
		// do something when the node is initially created
	},
	onInsert: function(){
		// do something each time the node is inserted into the DOM
	},
	events: {
		'click:delegate(ui-toggler)': function(event){
			// activate the clicked toggler
		}
	},
	methods: {
		getSelectedIndex: function(){
			// get the index of the selected toggler element
		},
		nextToggler: function(){
			// activate next toggler element
		},
		previousToggler: function(){
			// activate the previous toggler element
		}
	}
});

Tag Demos

Tabbox

When you need an advanced, interactive element for your app, like a map, tree list, or spin wheel input, you usually need to find a plugin, write a block of HTML, and instantiate the plugin using elements you retrieve from the DOM in your own JS. Let's get real: that sucks. What if all you did was drop a custom tag in the page and it Just Worked™? Well folks, x-tag turns that "what if" into reality.
